## 첫 사용 시 알아두기

### ⏱️ 소요 시간
- **처음 실행**: 3~5분 (패키지 설치 포함)
- **이후 실행**: 10~20초
- **분석 시작**: 10~30분 (종목 수에 따라)

### 📊 추천 프로세스

1. **"빠른 분석 (50개)"로 시작**
   - 50개 종목만 분석하여 빠르게 결과 확인
   - 시간: 약 5~10분

2. **상위 종목 차트 확인**
   - 📈 차트 분석 탭에서 추천 종목의 차트 확인
   - 기술적 상황 파악

3. **데이터 다운로드**
   - 📊 데이터 테이블 탭에서 CSV 다운로드
   - 엑셀 등에서 추가 분석

4. **전체 분석으로 업그레이드**
   - 충분한 검토 후 전체 종목 분석 실행
   - 최적의 종목 발굴

### ⚠️ 중요한 주의사항

1. **인터넷 연결 필수**
   - 실시간 주식 데이터를 받아옵니다

2. **처음 분석은 시간이 걸립니다**
   - 전체 2,000개 이상 종목 분석: 20~30분
   - 이후 재분석은 더 빠를 수 있습니다

3. **투자는 본인 책임**
   - 이 도구는 참고용입니다
   - 추천이 100% 정확하지 않습니다
   - 충분한 검토 후 투자 결정하세요

## 문제 해결

### Q: "Python을 찾을 수 없습니다" 에러
**A**: Python3 설치 확인
```bash
python3 --version
```

### Q: 처음 실행이 너무 오래 걸려요
**A**: 정상입니다. 첫 실행 시 패키지 설치 때문에 3~5분 소요

### Q: 분석 도중 멈춤
**A**: 네트워크 연결 확인 후 다시 시도

### Q: 다른 포트에서 실행하고 싶어요
```bash
streamlit run app.py --server.port 8502
```
